# Week-3-Project-School-Type-Academic-Performance-in-Catalonia



\# 📊 School Type \& Academic Performance in Catalonia



A data wrangling and exploratory data analysis (EDA) project investigating the relationship between \*\*school type\*\*, \*\*student enrolment\*\*, and \*\*academic performance\*\* in Catalonia over four academic years (2020/2021–2023/2024).



\---



\## 📌 Project Overview



This project explores whether students from \*\*public\*\* and \*\*private\*\* schools differ in their academic performance using official data from Catalonia.



The analysis combines university entrance examination (PAU) results with regional student enrolment statistics through an end-to-end data wrangling pipeline, including data collection, cleaning, transformation, integration, and visualization.



\---



\## 🎯 Objectives



\* Compare average PAU scores between public and private schools.

\* Analyse performance trends over four academic years.

\* Compare enrolment patterns by school type.

\* Explore the relationship between student enrolment and academic performance.

\* Demonstrate a complete data wrangling workflow using multiple real-world datasets.



\---



\## 📂 Data Sources



\### 1. PAU Results



Official PAU (University Entrance Examination) results published by the Generalitat de Catalunya.



Contains:



\* Average PAU scores

\* School type (Public / Private)

\* Administrative education region

\* Academic year



\### 2. Student Enrolment



Official student enrolment statistics published by Idescat.



Contains:



\* Number of enrolled students

\* School type

\* Region

\* Academic year



\---



\## 🛠 Data Wrangling Process



The project involved several preprocessing steps:



\* Importing multiple CSV and Excel datasets

\* Handling encoding and delimiter inconsistencies

\* Cleaning missing and duplicated values

\* Standardising column names

\* Mapping Catalan counties (Comarques) to Territorial Education Regions

\* Reshaping data into analysis-ready format

\* Merging PAU and enrolment datasets

\* Creating aggregated regional indicators



\---



\## 📈 Analysis



The project includes:



\* Descriptive statistics

\* Comparative analysis (Public vs. Private)

\* Trend analysis across four academic years

\* Regional comparisons

\* Relationship analysis between enrolment and academic performance

\* Data visualisation using Matplotlib and Pandas



\---



\## 📊 Key Findings



\* Private schools consistently achieved higher average PAU scores than public schools.

\* Both school types showed a slight decline in average scores over the four-year period.

\* Public schools enrolled substantially more students than private schools.

\* Higher student enrolment did not necessarily correspond to better academic performance.



\---



\## ⚠ Limitations



\* Statistical significance testing (e.g., t-tests) was not performed.

\* Socioeconomic factors were not included.

\* The analysis is based on aggregated regional data rather than individual student records.

\* Only four academic years were analysed.
